# IMEpay

**imepay** Task application.

## Features

- **Carousel Slider**: Display a carousel of items with smooth transitions.
- **Connectivity Plus**: Monitor network connectivity and handle offline scenarios.
- **Dio**: Manage HTTP requests with powerful capabilities.
- **Equatable**: Simplify value comparisons in data classes.
- **Floor**: Local database access using a type-safe and reactive API.
- **Flutter BLoC**: Implement the BLoC pattern for state management.
- **Get It & Injectable**: Dependency injection for better code modularity and testability.
- **URL Launcher**: Launch URLs in a mobile application.
